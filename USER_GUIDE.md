# دليل استخدام موقع SaberKart

SaberKart هو نموذج واجهة متجر إلكتروني عربي مستوحى من طريقة تنظيم المتاجر الكبيرة مثل Flipkart، لكنه يستخدم اسمًا وهوية مستقلة ولا يحتوي على نسخ مباشر للشعارات أو العلامات التجارية.

## ما الذي يحتويه الموقع؟

- **شريط علوي ثابت:** يحتوي على اسم المتجر، مربع البحث، تسجيل الدخول، رابط البائعين، الدعم، وسلة التسوق.
- **شريط التوصيل:** يعرض اختيار موقع التوصيل، التوصيل المجاني، والدفع الآمن.
- **أقسام سريعة:** روابط للأقسام المهمة مثل الهواتف، الإلكترونيات، الأزياء، المنزل، الأجهزة، الجمال، الكتب، والرياضة.
- **واجهة ترحيبية كبيرة:** تعرض عرضًا مميزًا وزرين للتنقل إلى العروض أو دليل الاستخدام.
- **بطاقات عروض:** ثلاث بطاقات تسويقية للأجهزة، الموضة، والمنزل الذكي.
- **قائمة منتجات تفاعلية:** تعرض منتجات مع السعر، التقييم، وزر الإضافة إلى السلة.
- **فلترة وبحث:** يمكن اختيار قسم محدد أو البحث باسم المنتج ووصفه.
- **قسم خدمات:** يعرض الدعم، عدد المنتجات التجريبي، سياسة الاسترجاع، ومعلومة للبائعين.

## طريقة الاستخدام للزائر

1. افتح ملف `index.html` في المتصفح أو شغّل خادمًا محليًا.
2. استخدم مربع البحث أعلى الصفحة للبحث عن منتج مثل `Saber` أو `لابتوب`.
3. اختر قسمًا من القائمة المنسدلة بجانب عنوان المنتجات لتصفية النتائج.
4. اضغط زر **أضف إلى السلة** داخل أي بطاقة منتج، وسيزداد رقم السلة في الشريط العلوي.
5. استخدم روابط الأقسام للانتقال داخل الصفحة بسرعة.

## طريقة التشغيل محليًا

يمكن تشغيل الموقع كملفات ثابتة مباشرة، أو عبر خادم Python بسيط:

```bash
python3 -m http.server 8000
```

ثم افتح الرابط التالي في المتصفح:

```text
http://localhost:8000
```

## الملفات المهمة

- `index.html`: هيكل الصفحة وكل الأقسام الرئيسية.
- `styles.css`: التصميم، الألوان، التخطيط، والاستجابة للشاشات الصغيرة.
- `script.js`: المنتجات التجريبية، البحث، الفلترة، ومنطق السلة.
- `USER_GUIDE.md`: هذا الدليل.

## كيف تطوّر الموقع لاحقًا؟

- إضافة صفحة منتج تفصيلية لكل منتج.
- ربط السلة بقاعدة بيانات أو LocalStorage لحفظ اختيارات المستخدم.
- إضافة تسجيل دخول حقيقي للمستخدمين.
- إضافة بوابة دفع مثل Stripe أو PayPal.
- إنشاء لوحة تحكم للبائعين لإضافة المنتجات وتعديل الأسعار.
- ربط المنتجات بواجهة API بدلًا من البيانات الموجودة داخل `script.js`.

## ملاحظة مهمة

هذا المشروع نموذج واجهة أمامية فقط. لا يعالج المدفوعات، ولا يخزن بيانات حقيقية، ولا يرسل طلبات شراء فعلية. قبل استخدامه تجاريًا يجب إضافة نظام خلفي آمن، قواعد بيانات، حماية حسابات، وسياسة خصوصية وشروط استخدام.
