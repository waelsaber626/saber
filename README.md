# SaberKart

SaberKart هو نموذج متجر إلكتروني عربي ثابت، مصمم كتجربة أولية قريبة من تنظيم متاجر التجارة الإلكترونية الكبيرة: شريط بحث، أقسام، عروض، منتجات، فلترة، وسلة تفاعلية بسيطة.

## التشغيل السريع

افتح `index.html` مباشرة في المتصفح، أو شغّل خادمًا محليًا:

```bash
python3 -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```

## الملفات

- `index.html`: بنية واجهة المتجر.
- `styles.css`: تصميم الواجهة والاستجابة للشاشات.
- `script.js`: بيانات المنتجات، البحث، الفلترة، والسلة.
- `USER_GUIDE.md`: دليل الاستخدام والإرشادات.
